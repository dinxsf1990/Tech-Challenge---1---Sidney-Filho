# Análise de E-commerce (Olist)

## Objetivo
Analisar crescimento, comportamento de consumo e eficiência logística.

## Dados
Brazilian E-Commerce Public Dataset by Olist

## Principais Insights
- Crescimento impulsionado por volume
- Concentração geográfica em SP
- Diferenças regionais de consumo
- Logística estável

## Ferramentas
- Python
- Pandas
- Seaborn
- Matplotlib
